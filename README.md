# Google Landing Page Clone

A simple Google search page copy made with HTML and CSS. It has a cool dark theme!

## What's this?

This is mine Assignment-0 I recreated the Google search homepage. It looks like the real Google but with a dark mode design. Everything is built from scratch using plain HTML and CSS.

## Files in this project

**index.html** - The structure of the page
- Header with links (About, Store, Gmail, Images)
- Big Google logo in the middle
- Search bar with microphone and image search icons
- Footer with language options

**index.css** - All the styling
- Dark background (#212227 - looks sleek!)
- White text that's easy to read
- Flexbox layout for everything
- Nice hover effects on buttons and links
- Mobile-friendly responsive design

## How to use it

Just open the `index.html` file in your browser and you'll see the Google homepage clone!

## Features

Dark theme - easier on the eyes  
Centered search bar with icons  
Works on different screen sizes  
Hover effects on all clickable stuff  
Clean and simple design  

## What you need

Any modern web browser (Chrome, Firefox, Safari, Edge, etc.) - that's it!

## CSS Units Explained (px, rem, em)

**px (pixels)** - Fixed size units that don't change based on screen size. Used for borders, icons, and small elements.  
**rem (root em)** - Relative units based on the root font size (usually 16px). Used for responsive designs so everything scales together.  
**em** - Relative units based on the parent element's font size. Used for padding/margins inside specific components so they scale with the text.
